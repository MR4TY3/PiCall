PiCall — Бесплатные звонки и чат

PiCall — это защищённый веб-мессенджер с голосовыми и видеозвонками. Работает прямо в браузере, ничего качать не нужно. Но если хочется — есть приложение для Windows.

Сайт: picall.duckdns.org:8000

---

Возможности

Чаты

- Общий чат — общайтесь со всеми пользователями PiCall в реальном времени
- Личные сообщения — найдите любого пользователя по нику или Pi/ID и начните приватный диалог
- Поиск — быстрый поиск по всем пользователям
- Удаление чатов — удаляйте ненужные диалоги; если ошиблись — есть 5 секунд на отмену

Звонки (WebRTC)

- Голосовые звонки — звоните другим пользователям прямо из браузера
- Видеозвонки — включайте камеру и общайтесь лицом к лицу
- Переключение камеры — фронтальная / тыловая (на мобильных)
- Управление микрофоном — включить / выключить звук во время звонка
- Управление видео — включить / выключить трансляцию с камеры
- Входящие звонки — модальное окно с уведомлением о входящем вызове

Безопасность

HTTPS, 2FA, капча, защита от XSS, CSRF, брутфорса и DoS — ваши данные под надёжной защитой.

Профиль

- Никнейм — может быть любым, даже совпадать с другими пользователями
- Pi/ID — уникальный идентификатор вида pi/имя (3–10 символов)
- Аватар — автоматически генерируется из первой буквы ника
- Копирование Pi/ID — одним нажатием скопировать свой ID в буфер обмена

Интерфейс

- Тёмная тема — стильный тёмный дизайн, приятный для глаз
- Анимация частиц — красивая фоновая анимация на десктопе (автоматически отключается на мобильных для экономии батареи)
- Адаптивный дизайн — одинаково удобно на телефоне, планшете и компьютере
- SVG-иконки — все иконки векторные, выглядят чётко на любом экране
- Плавные анимации — открытие окон, переходы между состояниями

---

Как начать пользоваться

1. Откройте сайт: picall.duckdns.org:8000
2. Зарегистрируйтесь: придумайте имя пользователя и пароль (пароль должен быть достаточно сложным)
3. Настройте профиль: укажите никнейм и уникальный Pi/ID
4. Начните общаться: пишите в общий чат или находите других пользователей через поиск
5. Звоните: нажмите на кнопку звонка в личном чате — собеседник получит уведомление

Совет: включите двухфакторную аутентификацию в настройках профиля — это надёжно защитит ваш аккаунт.
